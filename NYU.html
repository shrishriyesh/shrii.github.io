Projects

Learned Index Structures: Implemented a Pytorch-based system to create learned index structures, optimizing database indexing using deep learning to efficiently position records and reduce memory consumption. The model was validated through benchmarking experiments to ensure ultra-low latency and robustness.

Replicated Distributed DB Implementation with Serializable Snapshot Isolation: Simulated a multi-site distributed database system ensuring data consistency and supporting concurrent transactions. Implemented replication and fault tolerance to handle site failures and recoveries, enabling the system to maintain operations even during partial outages.

Multimodal Music Generation Pipeline: Developed a pipeline for music generation from multimodal inputs using REST APIs. Integrated an auto-regressive Transformer model (MusicGen) with retrieval-augmented generation, and implemented a MongoDB Atlas-based data storage to support interactive user-generated music outputs.

GPU Accelerated Vector Database Querying: Designed a CUDA-based GPU-accelerated vector querying method to perform nearest-neighbor searches on a 20GB dataset containing 6.1 million+ articles. Achieved over 40% speedup by implementing dynamic kernel modes and optimized fine-grained search mechanisms.

Retail Market Analysis: Conducted comprehensive market basket analysis on retail transaction data to uncover customer purchasing patterns and product associations. Implemented Apriori and FP-Growth algorithms to identify frequent itemsets and optimized recommendation generation. Developed interactive dashboards using Python and Plotly to visualize trends and actionable insights, supporting data-driven marketing and inventory decisions.

Urban Dynamics Data Analysis Project: Analyzed 10 million records of urban data, including crime rates, housing prices, and traffic patterns for Chicago. Utilized Hadoop and Presto for data processing and leveraged Tableau for creating dynamic dashboards to uncover correlations and support data-driven policy decisions.

Big Judges, Small Solvers — Verifiable Reward Learning : Fine-tuned 3–7 B-parameter language models with PPO/GRPO while a larger AceMath-7B judge delivered Likert-scale rewards for solution correctness and reasoning quality on GSM-8K and MATH benchmarks, yielding up to 8 pp accuracy gains over a supervised baseline with quantized judge inference on commodity GPUs.
